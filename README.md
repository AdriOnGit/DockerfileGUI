# DockerFile-GUI

## Per eseguire:

1. Clonare la repository sulla macchina host.
2. Rimuovere i container che utilizzano l'IP `172.21.0.2`.
3. Eseguire il comando `docker-compose up -d --build` nella directory in cui si trovano i file `Dockerfile` e `docker-compose.yml`.
4. Caricare il sito sul browser.
